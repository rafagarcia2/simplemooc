DOCUMENTAÇÃO CURSO DJANGO - UDEMY

- USAR VIRTUALENV COM PYTHON 3:
	virtualenv -p /usr/python3 env4simplemooc
	source env4simplemooc/bin/activate

- INICIAR O PROJETO DJANGO
	django-admin startproject simplemooc

BANCO DE DADOS: sqlite

- SINCRONIZANDO O BANCO:
	python manage.py migrate


INICIAR APLICAÇÃO:
	python manage.py startapp core

	* A pasta da aplicação foi movida para 'simplemooc'; 
	* Configuração no APPS do setting.

	* Criação da URL e do method 'home';

TEMPLATES:
	* São criados na pasta template e renderizados na view.

	* Os arquivos html, ficam na própria pasta html, já os css ficam na pasta static.
	* A manipulação do HTML é operada por tags do Django: {{% block content %}}

	* A utilização do Django para Templates é bem poderosa.
